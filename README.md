# IRONLOOP AI 🚀

> AI-powered study operating system for disciplined students.

IRONLOOP AI is a modern AI-driven study platform designed for students preparing for CBSE boards, JEE, NEET, SAT, Olympiads, and competitive examinations. The platform combines intelligent study planning, AI tutoring, mistake tracking, revision systems, analytics, and productivity tools into one unified ecosystem focused on mastery and continuous improvement.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/Optimus-Prime04/IronLoop-AI.git
```

## Install Dependencies

```bash
pnpm install
```

## Run Development Server

```bash
pnpm dev
```

---

# 🔐 Environment Variables

Create a `.env.local` file:

```env
NEXT_PUBLIC_SUPABASE_URL=your_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_key
OPENAI_API_KEY=your_openai_key
```
